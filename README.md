# Qaqash AI Chatbot

iOS app for Azerbaijan market.

## Setup

```bash
cd qaqash_app
flutter pub get
flutter build ios
```

## Requirements
- Flutter SDK
- Xcode (for iOS)
- OpenRouter API key

## Features
- AI Chat (Qwen/MiniMax)
- Azerbaijani language support
- Dark theme
- Voice output (TTS)

## App Store
- Bundle ID: com.qaqash.app
- Version: 1.0.0
